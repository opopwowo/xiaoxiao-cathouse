# 小小貓屋 — 本機開發環境設定（Windows）

在新電腦上要能編輯／預覽網站，照這份順序做即可。這個專案是**純靜態網頁 + Cloudflare Worker**，
沒有 `package.json`，本機預覽用 `npx wrangler dev`。

---

## 第 1 步：安裝三個必備軟體

到官網下載安裝，全部選預設值、一直「下一步」：

| 軟體 | 下載網址 | 用途 |
|------|---------|------|
| Git for Windows | https://git-scm.com/download/win | 版本控制、上傳到 GitHub |
| Node.js（LTS 版） | https://nodejs.org/ | 執行 Wrangler 本機預覽 |
| VS Code | https://code.visualstudio.com/ | 程式編輯器 |

> Node.js 選綠色的 **LTS** 按鈕（不要選 Current）。裝完把電腦**重開機一次**讓指令生效。

---

## 第 2 步：驗證安裝成功

開「開始 → 輸入 `cmd` → 命令提示字元」，一行一行貼上按 Enter：

```
git --version
node --version
npm --version
```

三行都要各自印出版本號。若有一行說「不是內部或外部命令」代表那個沒裝好。

---

## 第 3 步：設定 Git 身分（第一次才要做）

```
git config --global user.name "你的名字"
git config --global user.email "opopwowo@gmail.com"
```

---

## 第 4 步：下載專案到筆電

```
cd %USERPROFILE%\Documents
git clone https://github.com/opopwowo/xiaoxiao-cathouse.git
cd xiaoxiao-cathouse
```

第一次會跳出視窗要你登入 GitHub（用瀏覽器授權即可）。

---

## 第 5 步：用 VS Code 打開專案

```
code .
```

（最後有空格加句點）以後可直接在 VS Code 裡編輯檔案。

---

## 第 6 步：本機預覽網站

VS Code 選單「終端機 → 新增終端機」，貼上：

```
npx wrangler dev
```

第一次會問要不要安裝 wrangler，輸入 `y`。跑起來後打開顯示的網址（通常 `http://localhost:8787`）
即可預覽，改檔存檔後重整就更新。停止按 `Ctrl + C`。

---

## 上傳修改（標準三步驟）

```
git add .
git commit -m "說明改了什麼"
git push
```

---

## ⚠️ 重要提醒

1. **推上 `master` 分支 = 正式網站 `lovecat.cc` 會自動更新**（GitHub Actions 自動部署），確認改對再上傳。
2. **新增／下架貓咪要同步三個地方**（漏一個資料會不一致）：
   - `public/index.html` 的 `const kittens`
   - `public/kittens-data.js`
   - `src/worker.js` 的 `kittenMeta`
3. 貓咪照片放 `public/images/kitten-kXX.jpg`，`img` 欄位填一般網址，**絕對不要用 base64**。

更多專案規則見根目錄的 `CLAUDE.md`。
