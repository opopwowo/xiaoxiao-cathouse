## 品種介紹

### 基本資料

夏特爾貓（Chartreux）是源自法國的古老天然品種，被視為法國的「國寶貓」，傳說與法國加爾都西會（Carthusian）修道院淵源深厚。最大的特徵是一身均勻的藍灰色（藍色）短毛，搭配明亮的銅橘色或金黃色眼睛，以及天生嘴角微微上揚的「微笑臉」。

| 項目 | 內容 |
| --- | --- |
| 英文名 | Chartreux |
| 原產地 | 法國 |
| 體型 | 中至大型，肌肉結實 |
| 體重 | 公貓約 5–7 公斤、母貓約 3–5 公斤 |
| 毛長 | 短毛、雙層被毛、略帶羊毛質感 |
| 毛色 | 各種深淺的藍灰色（單一藍色） |
| 眼睛 | 銅橘色至金黃色 |
| 平均壽命 | 約 12–15 年 |

夏特爾貓骨架厚實、四肢相對較細，呈現出「火柴棒上的馬鈴薯」般可愛比例，是少數兼具力量與柔和氣質的品種。

### 個性

夏特爾貓以「安靜的陪伴者」著稱。牠們叫聲非常少，有些甚至幾乎不出聲，個性沉穩、溫和、忠誠，喜歡默默跟在主人身邊，是典型「狗狗性格」的貓。對家人黏人但不過度索求，對陌生人則保持禮貌的觀察距離。

牠們聰明且適應力強，能與兒童、其他貓狗和平共處，情緒穩定，很少出現破壞行為，非常適合上班族與重視居家寧靜的家庭。

### 外觀毛色

夏特爾貓的毛色只有一種——藍灰色，但深淺可從淺灰藍到偏深的石板灰。被毛為雙層結構，帶有些微防水的羊毛質感，摸起來厚實有彈性。鼻頭為石板灰、肉墊偏玫瑰灰，銅橘色的眼睛與藍灰毛色形成迷人對比。幼貓時期可能帶有淡淡的虎斑殘影，會隨成長逐漸消退為純色。

### 健康問題與基因檢測

夏特爾貓整體健康強健，但仍有幾項需留意的遺傳傾向：

- **髖關節發育不良（Hip Dysplasia）**：大型骨架品種較需注意，建議維持理想體重、避免肥胖。
- **PK-Def（丙酮酸激酶缺乏症）**：一種會造成貧血的遺傳性疾病，負責任的繁育者應對種貓進行基因篩檢。
