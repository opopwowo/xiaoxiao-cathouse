## 品種介紹

### 基本資料

玩具虎貓（Toyger，名稱取自 Toy + Tiger「玩具老虎」）是一個相當年輕的人工培育品種，於 1980 年代由美國育種者以 [孟加拉貓](/breed/bengal) 為基礎，搭配家貓選育而成，目標是打造出擁有「老虎般直立虎斑」的家貓外型。牠在 2007 年才獲得 TICA 正式認可，是貓界中相當新的品種。

| 項目 | 內容 |
| --- | --- |
| 英文名 | Toyger |
| 原產地 | 美國 |
| 培育來源 | 以孟加拉貓改良選育 |
| 體型 | 中型，肌肉結實 |
| 體重 | 公貓約 5–7 公斤、母貓約 3.5–5 公斤 |
| 毛長 | 短毛、柔順有光澤 |
| 毛色 | 橘金底色配深色直立虎斑（brown mackerel tabby 為基礎改良） |
| 眼睛 | 圓而深邃 |
| 平均壽命 | 約 12–15 年 |

玩具虎貓最大的賣點是「縮小版老虎」的外觀，但個性卻是溫和親人的家貓，完全沒有野性難馴的問題。

### 個性

玩具虎貓承襲了孟加拉貓的高智商與活力，個性活潑、好奇、精力旺盛，同時又非常親人、樂於互動。牠們聰明到可以學會玩你丟我撿、定點如廁、甚至牽繩散步，喜歡跟隨主人、參與家庭活動，是會「黏著你一起生活」的貓。

牠們適應力佳、不怕生，與兒童及其他寵物相處融洽，但因活力旺盛，需要足夠的遊戲與環境刺激來消耗體力。

### 外觀毛色

玩具虎貓的標誌是「垂直、分支狀的深色直紋虎斑」，相較於一般虎斑貓的橫向紋路，玩具虎貓追求的是模仿老虎的直立條紋，底色則為溫暖的橘金色至棕褐色，部分個體還帶有「亮片般（glittering）」的毛尖光澤——這也是源自孟加拉貓的特徵。臉部紋路呈圓形向心排列，營造出威武又可愛的「小老虎臉」。

### 健康問題與基因檢測

玩具虎貓整體健康，但因培育自孟加拉貓血系，需留意源自家貓常見的遺傳傾向：

- **肥厚性心肌病（HCM）**：貓科常見心臟病，建議選擇有心臟篩檢紀錄的種貓。
- **丙酮酸激酶缺乏症（PK-Def）**：可能造成貧血的遺傳病，負責任的繁育者會做基因檢測。
