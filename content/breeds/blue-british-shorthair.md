## 品種介紹

### 基本資料

藍貓英短（Blue British Shorthair）是[英國短毛貓](/breed/british-shorthair)最經典、最具代表性的色系，也是許多人心目中「英短」的標準形象。牠並非另一個品種，而是英短的純藍灰色版本。穩重的個性、福氣的圓臉與厚實身軀，讓藍貓多年來穩坐人氣榜前列。

| 項目 | 內容 |
| --- | --- |
| 英文名 | Blue British Shorthair |
| 體型 | 中大型，肌肉結實 |
| 體重 | 公貓 4–8 公斤、母貓 3–5 公斤 |
| 毛長 | 短毛（亦有藍色英長） |
| 眼睛顏色 | 銅橘色／金橘色 |
| 平均壽命 | 12–17 歲 |
| 個性關鍵字 | 穩重、獨立、黏人、親人 |

### 個性

藍貓英短是公認的「紳士貓」代表：性格沉穩、情緒平穩、不愛吵鬧。牠們黏人卻保有界線，喜歡在你身邊趴著陪伴，而非整天討抱。對環境與家庭成員適應力強，對孩童、長輩與其他寵物都相當友善，是少數幾乎不挑飼主的貓種，因此特別受新手歡迎。

### 外觀毛色

藍貓最鮮明的辨識特徵，是**整身均勻、無雜色的灰藍色短毛**——沒有漸層、沒有條紋，每一處顏色一致，毛質短而濃密、觸感像絨毛。搭配一雙又大又圓的**銅橘色眼睛**，與漸層英短的綠眼形成強烈對比，這正是分辨藍貓與金／銀漸層最快的方法。再加上英短標誌性的圓臉、飽滿臉頰、短鼻與圓耳，整體圓潤討喜，福相十足。

藍貓短毛濃密，一週梳 2–3 次可有效減少掉毛與毛球，換毛季可增加頻率。藍貓食慾旺盛、又是出了名的易胖體質，務必定量餵食、避免放飯式餵養，並用逗貓棒等互動維持活動量。牠們天生慵懶、運動意願低，體態管理是日常照顧的重中之重。
