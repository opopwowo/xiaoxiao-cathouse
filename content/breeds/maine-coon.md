## 品種介紹

### 基本資料

緬因貓（Maine Coon）原產於美國緬因州，是世界上體型最大的家貓品種之一，有「溫柔的巨人」之稱。牠們擁有壯碩的骨架、豐厚的長毛、帶簇毛的大耳朵與蓬鬆的長尾巴，外型威武卻個性溫馴。

| 項目 | 內容 |
| --- | --- |
| 英文名 | Maine Coon |
| 體型 | 大型 |
| 平均體重 | 公貓 6–9 公斤、母貓 4.5–6.5 公斤 |
| 平均壽命 | 約 12–15 年 |
| 被毛 | 厚重長毛、防水質地 |
| 個性關鍵字 | 溫和、聰明、親人、愛互動 |

緬因貓發育較慢，通常要 3–4 歲才完全成熟。

### 個性

緬因貓個性溫和友善、聰明且非常親人，常被形容為「狗系貓」，會跟著主人、回應呼喚，甚至能學會簡單把戲。牠們愛玩水、好奇心強、叫聲特別（常發出可愛的啾啾聲而非標準喵叫）。對小孩與其他寵物都很包容，是極佳的家庭伴侶貓。

### 外觀毛色

緬因貓最具辨識度的就是龐大壯碩的身軀、方形吻部、帶簇毛（猞猁耳尖）的大耳朵，以及如旗幟般蓬鬆的長尾巴。被毛厚重防水，胸前常有豐厚毛領圈。毛色花色非常多元，最經典的是棕色虎斑，另有銀虎斑、黑、白、玳瑁、三花等多種變化。

### 照顧重點

- **梳毛**：長毛厚重，建議每天或隔天梳理，換毛季更需勤梳避免打結。
- **空間與運動**：體型大、活動量足，需要寬敞空間與堅固的大型跳台。
- **飲食**：成長期與成貓需足夠且均衡的營養支持骨架與肌肉。
- **健康檢查**：定期健檢，留意心臟與關節（詳見下節）。

### 健康問題與基因檢測

緬因貓作為大型貓，有幾項健康重點需特別留意：

- **肥厚性心肌病（HCM）**：是緬因貓較常見的遺傳性心臟病，負責任的繁殖會對種貓進行 HCM 相關基因檢測與心臟超音波篩檢。
- **髖關節發育不良**：大型品種較容易出現，需維持理想體重、避免過度肥胖以減輕關節負擔。
- 其他如脊髓性肌肉萎縮（SMA）等也有相關基因檢測可篩檢。
