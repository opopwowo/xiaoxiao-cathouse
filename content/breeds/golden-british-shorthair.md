## 品種介紹

### 基本資料

金漸層英短（Golden Shaded British Shorthair）並不是一個獨立的貓種，而是[英國短毛貓](/breed/british-shorthair)眾多色系中極受歡迎的一支。牠的骨架、臉型、個性都和傳統英短一致，唯一的差別在於毛色——底層為金黃色，毛尖暈染棕黑，呈現出層次分明的「漸層」效果。以下是基本資料概覽。

| 項目 | 內容 |
| --- | --- |
| 英文名 | Golden Shaded British Shorthair |
| 體型 | 中大型，肌肉結實 |
| 體重 | 公貓 4–7 公斤、母貓 3–5 公斤 |
| 毛長 | 短毛（亦有長毛金漸層） |
| 眼睛顏色 | 綠色或琥珀色 |
| 平均壽命 | 12–16 歲 |
| 個性關鍵字 | 穩重、獨立、黏人、安靜 |

### 個性

金漸層英短承襲了英短「紳士貓」的經典性格：穩重、不吵鬧、情緒平穩。牠們黏人卻不黐人，喜歡待在你身邊的同一個空間，卻未必要被抱在懷裡。這種「陪伴但保有距離」的相處方式，特別適合上班族與第一次養貓的新手。金漸層通常溫和好相處，對其他寵物與孩童的包容度高，是公認的家庭友善貓。

### 外觀毛色

金漸層最迷人的辨識特徵，就是那身會「發光」的毛色。每一根毛髮從根部的暖金黃逐漸過渡到尖端的棕黑色，光線一打便折射出金棕光澤。臉部、四肢與背線顏色較深，腹部與下巴顏色最淺。眼周、唇線常有如描過眼線般的深色勾邊，搭配翠綠到琥珀色的眼睛，更顯貴氣。配上英短標誌性的圓臉、短鼻、圓耳與厚實身軀，整體像一顆溫暖的金色毛球。

### 照顧重點

短毛的金漸層日常梳理不費力，一週梳 2–3 次即可維持毛色光澤、減少掉毛與毛球。重點反而在「體態管理」：英短系天生易胖，金漸層也不例外，需定量餵食、避免無限放糧，並安排逗貓棒互動維持活動量。牠們不愛激烈運動，主人要主動引導。此外英短淚腺與鼻腔結構偏短，留意眼周清潔即可。

### 健康問題與基因檢測

英短色系普遍須注意 **多囊腎（PKD）** 與 **肥厚性心肌病（HCM）**，以及因易胖衍生的關節與心血管負擔。負責任的貓舍會針對種貓進行 PKD 基因檢測，避免將帶因個體用於繁殖。小小貓屋在台中以非籠飼家庭式環境飼養，幼貓出售前完成疫苗與驅蟲，並提供 **180 天健康保固**，讓新手安心。
