## 品種介紹

### 基本資料

銀漸層英短（Silver Shaded British Shorthair）和金漸層一樣，並非獨立貓種，而是[英國短毛貓](/breed/british-shorthair)的銀色漸層色系。牠的體型、臉型與性格都與傳統英短一致，差別在毛色——純白底毛搭配銀灰色毛尖，再配上一雙標誌性的翠綠眼，整體呈現清冷又高貴的氣質。

| 項目 | 內容 |
| --- | --- |
| 英文名 | Silver Shaded British Shorthair |
| 體型 | 中大型，骨架厚實 |
| 體重 | 公貓 4–7 公斤、母貓 3–5 公斤 |
| 毛長 | 短毛（亦有長毛銀漸層） |
| 眼睛顏色 | 翠綠色為主 |
| 平均壽命 | 12–16 歲 |
| 個性關鍵字 | 穩重、安靜、黏人、優雅 |

### 個性

銀漸層英短的個性與整個英短家族同出一轍：沉穩、不吵鬧、情緒穩定。牠們喜歡用「待在你附近」的方式表達依賴，而非黏在身上撒嬌，是低存在感卻又默默陪伴的療癒型貓咪。對陌生環境適應力佳，對孩童與其他寵物友善，非常適合忙碌的上班族與第一次養貓的人。

### 外觀毛色

銀漸層最大的辨識特徵，是「白底＋銀灰毛尖」的冷色調漸層。毛根接近純白，僅毛尖約八分之一染上銀灰，整體看起來像撒了一層銀霜，乾淨明亮。眼周、唇線與鼻緣常有黑色勾邊，如同天生畫了眼線，搭配一雙翠綠到黃綠色的眼睛，顯得格外靈動。配上英短圓臉、短鼻、圓耳的福相，銀漸層在「貴氣」與「可愛」之間取得了絕妙平衡。

### 照顧重點

短毛銀漸層日常梳理輕鬆，一週梳 2–3 次能維持毛色乾淨、減少掉毛。由於底毛偏白，眼周與下巴需留意清潔，避免淚痕與口水漬影響外觀。和所有英短一樣，銀漸層也容易發胖，需定量餵食並安排互動運動。牠們運動慾不高，主人要主動用逗貓棒引導活動，維持理想體態。

### 健康問題與基因檢測

英短色系共通需注意 **多囊腎（PKD）** 與 **肥厚性心肌病（HCM）**，以及肥胖帶來的關節、心血管風險。專業貓舍會對種貓進行 PKD 基因篩檢，杜絕帶因個體繁殖。小小貓屋位於台中，採非籠飼家庭式環境飼養，幼貓出售前完成疫苗與驅蟲，並提供 **180 天健康保固**，讓飼主接手後更安心。
